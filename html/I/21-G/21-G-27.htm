TITLE I
THE STATE AND ITS GOVERNMENT

CHAPTER 21-G
ORGANIZATION OF EXECUTIVE BRANCH

Code of Ethics

Section 21-G:27

    21-G:27 Supplemental State Agency Ethical Codes. – In addition to this code, each agency may promulgate a supplemental ethics code to address issues specific to that agency. In the event of a conflict with the provisions of this code, a stricter provision of an agency code shall govern. To the extent that this code or an ethics code adopted by an agency shall apply to classified employees, this code, or an agency code, shall be interpreted to be consistent with the provisions of the classified employees' collective bargaining agreement and the state personnel rules.

Source. 2004, 214:1. 2006, 21:2, eff. June 2, 2006.

Term weighting

You can weight query terms for relative importance by using the {weight} tag to affect the ranking of results. This tag appears before the query term and has no closing tag.

{weight value = n} query term

The value parameter specifies the relative weight assigned to the query term. This value can range from 0.0 to 1.0.

Consider the following query with weighting.

{weight value=.250}dog AND {weight value=.500}cat AND {weight value=1.000}pig

Documents that contain the word "pig" would be ranked higher than those that contain the word "cat," which, in turn, would be ranked higher than those that contain the word "dog."

If you do not specify a weight, Indexing Service uses 1.0 by default. There is no short-form expression for weighting and no close tag is specified or required.
